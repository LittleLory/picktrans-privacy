拾译（PickTrans）隐私政策

最后更新日期：2026年3月5日

核心承诺：拾译是一款完全本地运行的翻译工具，不收集、不存储、不传输您的任何个人数据。

1. 数据收集（Data Collection）

我们不收集任何用户数据。

拾译不会收集以下任何类型的数据：

  • 个人身份信息（姓名、邮箱、电话等）
  • 浏览历史记录
  • 翻译内容或文本
  • 网站访问记录
  • 设备信息
  • 位置信息
  • 使用统计数据
  • Cookies 或其他追踪技术

2. 数据处理（Data Processing）

所有数据处理均在本地完成。

当您使用拾译进行翻译时：

  • 选中的文本仅在您的浏览器本地处理
  • 翻译请求直接发送到您选择的翻译服务提供商（如 Google Translate、DeepL 等）
  • 我们不会拦截、记录或修改任何翻译内容
  • 用户设置（如语言偏好）仅存储在浏览器本地存储中

3. 数据存储（Data Storage）

我们不在服务器上存储任何用户数据。

拾译使用浏览器的本地存储功能（chrome.storage.local）来保存：

  • 用户偏好设置（源语言、目标语言）
  • 扩展配置选项

这些数据：

  • 仅存储在您的设备上
  • 不会同步到任何服务器
  • 可以通过卸载扩展完全删除

4. 数据共享（Data Sharing）

我们不与任何第三方共享用户数据。

拾译不会：

  • 向任何第三方出售、出租或交易用户数据
  • 与广告商共享数据
  • 与分析服务共享数据
  • 向开发者或运营者传输任何使用数据

注意：当您使用翻译功能时，选中的文本会直接发送到您选择的翻译服务提供商（如 Google、DeepL 等）。这些第三方服务有各自的隐私政策，请参阅：

  • Google Translate: https://policies.google.com/privacy
  • DeepL: https://www.deepl.com/privacy

5. 权限说明（Permissions）

拾译请求以下浏览器权限，仅用于提供翻译功能：

  • activeTab：读取当前标签页中选中的文本
  • storage：在本地保存用户设置
  • contextMenus：在右键菜单中添加翻译选项

这些权限不会被用于收集或传输任何个人数据。

6. 数据安全（Data Security）

由于拾译不收集或存储任何用户数据，因此不存在数据泄露风险。所有处理均在您的设备本地完成。

7. 儿童隐私（Children's Privacy）

拾译不会有意收集 13 岁以下儿童的任何信息。由于我们不收集任何数据，本扩展可以安全地供所有年龄段用户使用。

8. 隐私政策变更（Changes to Privacy Policy）

如果我们对隐私政策进行重大更改，将通过扩展更新说明通知用户。继续使用扩展即表示您接受更新后的政策。

Privacy Policy (English Version)

Last Updated: March 5, 2026

Core Commitment: PickTrans is a fully local translation tool that does not collect, store, or transmit any personal data.

1. Data Collection

We do not collect any user data.

PickTrans does not collect:

  • Personal identification information
  • Browsing history
  • Translation content or text
  • Website visit records
  • Device information
  • Location data
  • Usage statistics
  • Cookies or tracking technologies

2. Data Processing

All data processing is done locally.

When you use PickTrans:

  • Selected text is processed only in your browser
  • Translation requests are sent directly to your chosen translation provider
  • We do not intercept, log, or modify any translation content
  • User settings are stored only in local browser storage

3. Data Storage

We do not store any user data on servers.

PickTrans uses browser local storage (chrome.storage.local) to save:

  • User preferences (source/target languages)
  • Extension configuration options

This data:

  • Remains only on your device
  • Is never synced to any server
  • Can be completely removed by uninstalling the extension

4. Data Sharing

We do not share user data with any third parties.

PickTrans will not:

  • Sell, rent, or trade user data
  • Share data with advertisers
  • Share data with analytics services
  • Transmit any usage data to developers

Note: When using translation features, selected text is sent directly to your chosen translation provider (Google, DeepL, etc.). These third-party services have their own privacy policies.

5. Permissions

PickTrans requests the following browser permissions:

  • activeTab: Read selected text in current tab
  • storage: Save user settings locally
  • contextMenus: Add translation option to context menu

6. Data Security

Since PickTrans does not collect or store any user data, there is no risk of data breach. All processing is done locally on your device.

7. Children's Privacy

PickTrans does not knowingly collect information from children under 13. Since we collect no data, this extension is safe for all ages.

8. Changes to Privacy Policy

We will notify users of significant privacy policy changes through extension update notes. Continued use indicates acceptance of the updated policy.
